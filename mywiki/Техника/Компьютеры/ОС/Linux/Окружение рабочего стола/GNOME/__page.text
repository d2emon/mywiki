'''Gnome 3.6 @Евангелие от Фея'''

Состоялся релиз десктоп-окружения Gnome 3.6. Заинтересовался я, что же в конце-то концов изменилось в нем. И установил я из старого проверенного PPA-репозитория:

@@
sudo add-apt-repository ppa:gnome3-team/gnome3
sudo apt-get update && sudo apt-get dist-upgrade
sudo apt-get install gnome-shell
@@

перезагружаемся. Выбираем Сессию Gnome. В результате 12.04 обновился только до Gnome 3.4.1. А вот результат из версии 12.10

@@
fey@ubuntu-business:~$ gnome-shell --version
[[<<]]GNOME Shell 3.6.1
@@


PS больше всего понравился экран входа пользователя а-ля Win8 и даже спорный Nautilus 3.6.1 неплохо выглядел.
