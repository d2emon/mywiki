The early SCI0 interpreter did not include special characters such as the accent grave, accent aigu, and the umlaut. A later SCI0 fork named S.old enabled the foreign language support.

!!! Media

All the SCI0/1 games were released on both 3.5" & 5.25" disks, except XMAS Card 1988, Astro Chicken & Fun Seeker's Guide, which were public domain. Later, SCI11 were released solely on 3.5" or CD-ROM.

All the games listed are disk versions unless noted else.

!!! Graphics

SCI0 games were mostly made for the 16 color EGA graphics adapter. Only the last SCI0 game (Jones in the fast lane) used a newer graphics engine and supported the 256 color VGA.
SCI1 games were released as two versions: 16 color EGA and 256 color VGA versions. The 16 color versions are much more "ugly" and also smaller in size, compared to their 256 color counterparts. This was done for people with CGA/EGA/TANDY videocards and a low-memory setup.
SCI11 ended this tradition by adding an EGA 640x200 graphics driver that dithered the pictures dynamically from their 256-color source.
SCI2 games supported high-resolution 640x480 SVGA graphics.
