!! Типы местности

!!! Арктика

Это холодные, покрытые льдами полярные области, почти непригодные для поддержания жизни.

!!! Болото

В болотах, находящихся в этих низких влажных землях, часто встречаются залежи руды. Вы можете также вырастить здесь немного продовольствия и сахарного тростника.

!!! Влажный лес

Это тропическая лесистая местность, хорошая для рубки леса, для добычи небольшого количества меха и для добычи руды. После вырубки становится Топью (хорошо для добычи руды).

!!! Горы

Это большие области гористой местности, трудной для путешествий, но в которой вероятны источники руды и серебра. В горах нельзя образовывать колонии. Здесь наиболее богаты месторождения серебра.

!!! Дождевой лес

Это тропические джунгли, хорошие для рубки леса, для производства небольшого количества сахара и для добычи руды. После вырубки становится Болотом (хорошо для добычи руды).

!!! Лиственный лес

Это умеренная лесистая местность, хорошая для добычи меха с помощью ловушек и для рубки леса. После вырубки становится Прерией (хорошо для выращивания хлопка).

!!! Луг

Плодородная почва в умеренных областях. Она идеальна для выращивания табака. Здесь вы можете также добывать продовольствие. Здесь наиболее обильно растет табак.

!!! Морской путь

Это часть открытого океана, которая ведет к стандартным морским маршрутам от Нового Света до Европы, и наоборот. Чтобы возвратиться в Европу, судну необходимо только встать на морской путь и затем двигаться на восток (покидая восточное побережье) или на запад (покидая западное побережье).

!!! Низкорослый лес

Это сухая бесплодная местность с рассеянной растительностью. В этих областях можно найти немного руды. После вырубки становится Пустыней (хорошо для добычи руды).

!!! Океан

Это широкая открытая водная поверхность. Он полезен для лова рыбы, особенно вдоль побережий, и в местах рыбного промысла.

!!! Прерия

Это относительно плоская, открытая земля, идеальная для производства хлопка. Вы можете также вырастить здесь зерновые продовольственные культуры. Здесь наиболее обильно растет хлопок.

!!! Пустыня

Эти засушливые, редкие области - трудные места для выращивания продовольствия (хотя это возможно). Однако, Вы можете иногда добыть здесь руду.

!!! Равнины

Это хорошие области для выращивания широкого разнообразия продовольствия. Также здесь вы можете вырастить немного хлопка. Здесь наиболее обильно растет продовольствие.

!!! Саванна

Богатая, влажная почва особенно пригодная для выращивания сахарного тростника. Эта земля также хороша для производства продовольствия. Здесь наиболее обильно растет сахар.

!!! Смешанный лес

Это умеренная лесистая местность, хорошая для добычи меха с помощью ловушек и для рубки леса. После вырубки становится Равнинами (хорошо для выращивания продовольствия).

!!! Таежный лес

Это холодная лесистая местность, хорошая для добычи меха с помощью ловушек и для рубки леса. После вырубки превращается в Тундру (хорошо для добычи руды).

!!! Топь

Влажная область с соленой почвой, где море граничит с сушей. Вы можете вырастить здесь немного продовольствия и в изобилии найти руду.

!!! Тропический лес

Это влажная лесистая местность, хорошая для рубки леса и для добычи небольшого количества меха. После вырубки становится Саванной (хорошо для выращивания сахара).

!!! Тундра

На этой довольно холодной, открытой земле можно производить минимальное количество продовольствия, а также заниматься другими мелкими видами сельского хозяйства. Однако, здесь часто находится руда.

!!! Хвойный лес

Это умеренная лесистая местность, хорошая для добычи меха с помощью ловушек и для рубки леса. После вырубки становится Лугом (хорошо для выращивания табака).

!!! Холмы

Область с небольшими подъемами, дающая свободный подход к руде. Вы можете также развивать здесь некоторые виды сельского хозяйства. Здесь наиболее богаты месторождения руды.
