# Человек на корабле измеряет глубину реки шестом — следовательно, не особо.
# На рисунке, судя по фрукту, изображено начало осени или конец лета — значит, на юг.
# Двое обедают, третий проверяет глубину реки, четвертый ловит рыбу.
# Как минимум, улитка, паук и журавли.
# Судя по тому, как вода расходится у носа парохода — вверх.
# На тарелке у ребят лежит дыня — значит, это август.
