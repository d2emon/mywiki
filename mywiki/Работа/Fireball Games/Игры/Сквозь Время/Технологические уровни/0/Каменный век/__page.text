!! Каменный век

!!! Оружие

# Безоружный
# Камни (метательное)
# Каменный нож
# Дубина
# Копье
# Большая дубина

!!! Часть 1

Узкая каменистая тропа. Поднимите камни. Нападает воин-медведь.  Поднимите каменный нож. Нападает еще один воин-медведь. 
Прийдете в пещеру тигра. Нападает саблезубый тигр. В пещере лежит груда костей.
Дальше путь лежит направо по узкому скальному карнизу. Сзади катятся камни. Слева на уступе стоит воин-медведь и кидает камни.
Идите по тропе вдоль скалы налево. Тропа идет вдоль обрыва. Нападет сильный воин-медведь с дубиной. Ему будет помогать девушка, бросающая камни. Поднимите дубину.
Левее тропы будет мини-портал.
Идите дальше по тропе. Будет ловушка в виде камня-маятника в сетке. Прийдете к пещере гориллы. Нападет горилла.
Идите по скальному карнизу дальше. Нападет воин-медведь с дубиной. Спереди будут катиться камни. Прийдете в большую пещеру.

!!! Часть 2

Вы в большой круглой пещеры с рисунками охоты на быков. Сверху бросает камни воин-медведь. В пещере лежит большая дубина. Карабкайтесь по стене пещеры к выходу.
Сверху вас будут поджидать два воина-медведя с дубинами. Идите налево по узкой скале. Нападет воин с копьем. Поднимите копье. Затем у входа в ущелье будет  еще один воин с копьем.
В ущелье нападают две обезьяны. Идите дальше к небольшому гроту, где на вас нападет горилла.
От грота идите по скальному уступу налево. Перепрыгните через пропасть.
Справа у поворота тропы будет мини-портал.
Идите по тропе направо. Нападут сильный воин с копьем и саблезубый леопард.
Спускайтесь по ступенькам вниз, на еще одну узкую скалу. Поднимите камень. Нападут сильный воин-медведь и сильный воин с копьем. Проходите в пещеру гигантского медведя.


!!! Враги

* Воин-медведь(1). Человек в шкуре медведя. Вооружен камнями.
* Саблезубый тигр(3). 
* Сильный воин-медведь с дубиной(2). Человек в шкуре медведя. Вооружен дубиной.
* Девушка(1). Девушка в шкуре. Вооружена камнями.
* Горилла(3).
* Воин-медведь с дубиной(1). Человек в шкуре медведя. Вооружен дубиной.
* Воин с копьем(1). Человек в набедренной повязке. Вооружен копьем.
* Обезьяна(2). Мелкая обезьяна.
* Сильный воин с копьем(2). Человек в набедренной повязке. Вооружен копьем.
* Саблезубый леопард(3). 
* Сильный воин-медведь(2). Человек в шкуре медведя. Вооружен камнями.
* Гигантский медведь(4).
