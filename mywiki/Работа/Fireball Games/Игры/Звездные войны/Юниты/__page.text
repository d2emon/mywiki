!! Юниты

!!! Линкор

Линкор представляет собой корабль большего размера. У каждого играющего в арсенале по десять кораблей этого класса. Линкор перемещается на три парсека за один ход. Атакует корабль противника по фронту, с фланга и с тыла.

!!! Разведкатер

Разведкатер обладает меньшей огневой мощью. Однако он более маневренный.

Разведывательный катер представляет собой корабль Меньшего размера. У каждого играющего в арсенале по пятнадцать кораблей этого класса. Разведкатер перемещается на четыре парсека за один ход. Атакует корабль противника с фланга и с тыла.
