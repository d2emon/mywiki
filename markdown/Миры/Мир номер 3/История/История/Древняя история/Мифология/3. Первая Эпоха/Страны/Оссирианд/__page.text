!! Оссирианд

Оссирианд (синд. край семи рек) — территория, существовавшая в Белерианде до начала Второй Эпохи. Название получила из-за семи рек, протекавших по этой территории — Гелион и шесть его притоков: Аскар, Талос, Леголин, Брильтор, Дуильвен, Адурант. С запада ограничивался рекой Гелион, с востока — Синими горами.
