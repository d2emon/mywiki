Кардолан


Одно из северных королевств Срединных Земель, образовавшееся после распада Арнора.

Граница Кардолана с Артэдайном проходила по реке Берендуин (Брендивайн) на западе. С востока граница проходила по реке Митейтель (Седонне), а на юге по реке Гватло. Северной границей Кардолана, с Рудауром, была Великая восточная дорога.
