# Battlefield 2 / Battlefield 3 - Оманский залив

![](./images/top-10-interesnyh-igrovyh-lokacij-9.jpg)

Лучшая карта для многопользовательского режима. Оманский залив стал настоящим раем для игроков, которым нравились бои с бронетехникой, танками, вертолетами и истребителями. Снайперам также есть чем заняться. Один из самых приятных моментов во всей игре – взлет на вертолете с авианосца, и успешная посадка на пляже.
