Функция "Мои Любимые" позволяет составлять сборники для каждого из наших дисков по своему вкусу. Списки этих сборников будут храниться на Вашем жёстком диске в системной папке Windows в файле FavoritList.ini.

Для открытия окна "Мои Любимые" нажмите кнопку на панели управления. Для закрытия окна нажмите кнопку повторно.

Для "перетаскивания" треков из основного окна в пакет "Мои Любимые" и их сортировки там, Вы можете пользоваться методом "drag & drop".

Также для добавления в пакет и удаления из него треков Вы можете использовать кнопки и соответственно.

Нажатие кнопки приведёт к удалению всех треков из пакета "Мои Любимые".

Проиграть пакет Вы можете либо нажав правой кнопкой мыши в окне "Мои Любимые", либо нажав кнопку на панели управления.
