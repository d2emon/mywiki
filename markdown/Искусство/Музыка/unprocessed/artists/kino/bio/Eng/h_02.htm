Reproduction modes

  • - Play all the disk tracks
  • - Play tracks of the album selected with cursor
  • - Play tracks marked with red tick (click blank to tick)
  • - Play "Favorites" packet
  • - Play every track for 7 seconds in selected reproduction mode (demo mode)
    •

    Control

  • - Open/close all folders (albums)
  • - Search the track by name or by text fragment
  • - Disconnect RMG Player and replay MP3 files under WinAmp control only
  • - Display/hide WinAmp
  • - Display/hide "Information" window
    •

    "Favorites" (songs)

  • - Display/hide "Favorites" window
  • - Add the highlighted track or the whole album to "Favorites" (one can simply drag-and-drop them into corresponding window)
  • - Remove the highlighted tracks from "Favorites"
  • - Clear "Favorites"
  • Note: move tracks in "Favorites" by drag-and-dropping them.

    • Right mouth button menu

    When right click mouse in RMG Player workspace the following menu shows up. It provides not only MP3 files playback mode, but also:

  • information about track from the corresponding MP3 file tag;
  • interface language choice (Russian or English);
  • RMG Player workspace font setup;
  • print disk content with tracks numeration for MP3 player;
  • saving of selected tracks on your hard disk.

    • Side and bottom RMG Player control bars

    This button click will divide RMG Player workspace into two parts: you can select the album in the first and the track in the second. Double click in separator will change vertical and horizontal division.


    One of the panels contains the player, and another shows both address and size of MP3 file of the selected track. Double left click mouse to switch the panels.
