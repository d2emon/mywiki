РИТМИЧЕСКИЕ РИСУНКИ

     Все мелодии, которые мы обычно поем, можно разбить на части, которые имеют одинаковую длительность. Такие части называют ТАКТЫ. Все мелодии имеют ритмический рисунок, определенную ритмичность чередования нот, причем этот рисунок на протяжении всей мелодии не изменяется. Главной чертой этого рисунка является РАЗМЕР - это количество наиболее ярких и энергичных нот в одном такте. Очень часто этот рисунок существует как бы сам по себе, только в голове музыканта, например, когда целый такт длится одна длинная нота или даже не нота, а пустота, пауза. Но ритм должен четко отсчитываться в памяти аккомпаниатора на протяжении всей мелодии.
     Главная задача аккомпанемента - выделить те моменты, когда ноты должны звучать ярко и выделенно, даже если в этот момент звучит пауза, то есть задать и поддержать ритм на протяжении всей мелодии. Вторая задача аккомпанемента - украсить мелодию, заполнить паузы, окружить мелодию звуками, которые подчеркнут ее настроение. Наиболее часто применяют РАЗМЕРЫ, которые мы будем называть так: "ДВА"-"ТРИ"-"ЧЕТЫРЕ"-"ШЕСТЬ" и означают они то самое, названое выше, количество наиболее ярких и энергичных нот в одном такте. Например, если шагают солдаты, то они идут на счет: "РАЗ-ДВА-ЛЕВОЙ-ПРАВОЙ" это и есть размер "ЧЕТЫРЕ" и если солдаты будут петь песню, то им, чтобы не сбиться с шага придется петь именно в этом размере. Есть такая песня: 
ВЫ СЛЫ -- ШИТЕ, ГРОХОЧУТ САПО----ГИ  пауза  пауза  пауза
     |       |       |      |     |    |      |      |
     раз     два    три   четыре  раз  два   три  четыре

И ПТИЦЫ   ОША---ЛЕЛЫ--Е   ЛЕ--ТЯТ     пауза  пауза  пауза
    |      |      |    |        |        |     |       |
    раз   два    три  четыре   раз     два   три    четыре
     Если в этой песне на каждый счет зазвучит аккорд, то получится самый простой вид аккомпанемента. Графически он может выглядеть так: 
ВЫ СЛЫ -- ШИТЕ, ГРОХОЧУТ  САПО----ГИ  пауза  пауза  пауза
    |       |       |      |       |    |      |      |
    ^       ^       ^      ^       ^    ^      ^      ^
    Ь       Ь       Ь      Ь       Ь    Ь      Ь      Ь

    1       2       3      4       1    2      3      4
     Все удары попадают на гласные звуки. Это потому, что согласные звуки петь нельзя, они не тянутся. Видно, что аккомпанемент повторяет одно и то же, как бы по кругу. Чтобы условно изобразить рисунок такого ритма, достаточно изобразить то, что постоянно повторяется: 
|                |
| ^   ^   ^   ^  |
| Ь   Ь   Ь   Ь  |
|                |
| 1   2   3   4  |
|                |
     Ритм 4 можно аккомпанировать ударами указательного пальца на счет 1-2-3-4 
| ЯЯЯЯЯ   ЯЯЯЯЯ   ЯЯЯЯЯ   ЯЯЯЯЯ  |
| Y Y Y   Y Y Y   Y Y Y   Y Y Y  |
|   X       X       X       X    |
|                                |
|   1       2       3       4    |
     Для этой же песни можно использовать очень простой и эффектный рисунок ритма, щипком трех тонких струн и басовой струны.
     Эти примеры даны только для того, чтобы показать, что рисунков ритма может быть много. Каждый прием нужно отрабатывать в очень медленном темпе, обязательно отсчитывая вслух или про себя ударные тактовые цифры, которые написаны под стрелками.
     Главное - не спешить! Сначала отработайте ритм на приглушенных струнах, без всяких аккордов. Потом возьмите простой аккорд, например E, или D, или C и тренируйтесь на одном аккорде. В аккорде не должны звучать плохо прижатые струны, ритм должен быть ровным, медленным и четким. Только после этого учитесь делать связки между аккордами, начиная их с аккордов открытых тональностей, например D-G-D-A и т.д.
     Дальше будут даны графические изображения наиболее часто применяющихся ритмических рисунков.
     Большинство ритмов 4 легко переводится в ритм 2, для этого нужно просто вдвое уменьшить рисунок. Так же легко ритм 3 переводится в ритм 6, поэтому будут даны основные изображения только ритмов 4 и 3.
